menu_book Dummy's Guide to TrackerPro

Follow these simple steps to get your tracking system running smoothly.

Step 1: Quick Add Nodes

Go to Config Setup and click Quick Add Node 3 times to create your anchors. Simply type directly into the table row to name them (e.g., "North Corner") and set their X/Y coordinates on your floor plan.

Example: MAC: NODE-1234 | X: 10 | Y: 10

Step 2: Flash Firmware

Click Package Generator on each node/tag. Copy the generated code into your Arduino IDE and upload it to your ESP32 devices.

  • Make sure WiFi credentials are correct in the code.
  • Each device must have a unique MAC address.

Step 3: Register Tags

Register your Asset Tags in the Asset Tags table. These are the items you want to track (e.g., Forklifts, Pallets).

Step 4: Watch Live

Go to Live Telemetry. Once nodes and tags are powered on, you'll see them appear on the map automatically!

Pro Tip: Simulation Mode

Don't have hardware yet? Run python backend/mock_telemetry.py to see the system in action with fake data!
